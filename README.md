# Authtoken

Simplified encrypted authentication tokens using JWE.

## Installation

The package can be installed by adding `authtoken` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:authtoken, "~> 0.1.0"}
  ]
end
```
